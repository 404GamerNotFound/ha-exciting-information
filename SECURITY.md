# Security Policy

## Supported Versions

We support the latest release on the `main` branch. If you find a security issue in an older version, please still report it so we can assess impact.

## Reporting a Vulnerability

Please report security vulnerabilities by using GitHub Security Advisories for this repository. If that is not available, open a GitHub issue with minimal details and request a private follow-up.

We aim to acknowledge reports within 7 days and will keep you informed as we investigate.
